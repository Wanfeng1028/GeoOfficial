## 问题

分析某城市过去若干年建设用地扩张范围、速率和主要转化来源。

## 输入

- Landsat / Sentinel 多时相影像
- 矢量行政边界
- 参考土地利用分类

## 方法与步骤

1. 建立项目并定义研究范围与时间窗口
2. 导入多时相影像并进行预处理
3. 运行变化检测和分类
4. 统计扩张面积与来源类型
5. 生成地图、图表和报告草稿

## 工具

- QGIS
- GDAL
- Google Earth Engine
- Python

## 输出

- 变化图
- 面积统计表
- 方法记录
- 报告草稿

## 限制

- 示例数据为公开影像，精度受云覆盖和时间分辨率限制
- 分类方法仅为基础示例，不代表最优精度
