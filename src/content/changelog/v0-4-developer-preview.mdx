## 概述

GeoWork v0.4 Developer Preview 将桌面工作台、运行时与地理空间 Worker
的基础能力整合到一个可用的开发版本中。本次更新聚焦于工作模式框架、
任务编排基础，以及 Python Geo Worker 对 QGIS / GDAL 的接入。

## 亮点

- 桌面工作台 Work / Code / Map 模式框架
- Go Runtime 任务编排基础
- Python Geo Worker 接入 QGIS / GDAL
- Skills 与 MCP 基础扩展机制

## 已知问题

- 部分自动化能力仍属于 Developer Preview
- macOS 与 Linux 暂未提供官方构建
- 报告自动生成需人工核对
