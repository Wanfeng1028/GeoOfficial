# GeoWork v0.4 Developer Preview

## 总览

v0.4 开发版完善桌面工作台、运行时和地理空间 Worker 的基础能力，并接入 Skills 与 MCP 扩展机制。当前版本不作生产可用承诺，macOS 与 Linux 构建尚未提供。

## 主要能力

- 桌面工作台 Work / Code / Map 模式框架。
- Go Runtime 任务编排基础。
- Python Geo Worker 接入 QGIS / GDAL。
- Skills 与 MCP 基础扩展机制。

## 已知限制

- 自动化能力仍属于 Developer Preview，结果需人工核对。
- macOS 与 Linux 暂未提供官方构建。
- 报告自动生成属于辅助能力。

## 反馈

可通过 GitHub Issues 反馈问题，Discussions 提出建议。不承诺商业支持或团队协作功能时间表。
